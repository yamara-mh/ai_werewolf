# 🐺 AI人狼

AIと一緒に電子掲示板スタイルで遊ぶ人狼ゲームです。
https://yamara-mh.github.io/ai_werewolf/

## 遊び方

1. ロビーページでプレイヤー名・人数・人狼人数・追加役職・AIモデルを設定してゲームを開始
2. 昼フェーズ：掲示板にコメントを投稿し、人狼を議論で追い詰める
3. 投票フェーズ：疑わしいプレイヤーに投票して処刑する
4. 夜フェーズ：役職に応じて行動する（占い・護衛・襲撃）
5. 村人陣営と人狼陣営のどちらかが勝利条件を満たすまで繰り返す

## 役職

| 役職 | 陣営 | 能力 |
|------|------|------|
| 👤 村人 | 村人 | なし。人狼を全員排除したら勝利 |
| 🔮 占い師 | 村人 | 毎夜1人を占い人狼か判定できる |
| 🪬 霊媒師 | 村人 | 会議前に吊った人物が人狼か知ることができる |
| 🛡️ 騎士 | 村人 | 毎夜自分以外の1人を護衛し、襲撃から守れる |
| 🃏 狂人 | 人狼 | 村人だが勝利条件が人狼と同じ |
| 🐺 人狼 | 人狼 | 毎夜1人を襲撃し、村人と同数になったら勝利 |
| 🤝 共有者 | 村人 | もう一人の共有者が誰か把握できる |
| 🐈 猫又 | 村人 | 吊られたら村人を、襲撃されると人狼をランダムに一人道連れにする |
| 🦊 妖狐 | 第三陣営 | ゲーム終了時に生存していれば他陣営の代わりに単独勝利となる。占われると死亡する |
| 🐺 白狼 | 人狼 | 人間判定される |

## AI設定

- OpenAI / Gemini API Key（任意）を設定するとAIが本格的に発言・推理します
- 未設定でもサンプル発言でゲームを進行できます

## ファイル構成

```
index.html          # ロビーページ
game.html           # ゲームページ（BBS UI）
css/style.css       # スタイル
js/config.js        # 役職・定数定義
js/game.js          # ゲーム状態管理
js/ai.js            # AIプレイヤーロジック
js/bbs.js           # BBS レンダリング
js/main.js          # ロビーエントリーポイント
js/controller.js    # ゲームコントローラー
.github/workflows/deploy.yml  # GitHub Pages デプロイ
```
