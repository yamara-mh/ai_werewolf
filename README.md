# 🐺 AI人狼

AIと一緒に電子掲示板スタイルで遊ぶ人狼ゲームです。GitHub Pages で公開されます。

## 遊び方

1. ロビーページでプレイヤー名・人数・人狼人数・追加役職・AIモデルを設定してゲームを開始
2. 昼フェーズ：掲示板にコメントを投稿し、人狼を議論で追い詰める
3. 投票フェーズ：疑わしいプレイヤーに投票して処刑する
4. 夜フェーズ：役職に応じて行動する（占い・護衛・襲撃）
5. 村人陣営と人狼陣営のどちらかが勝利条件を満たすまで繰り返す

| 🪬 霊媒師 | 村人 | 処刑者の役職が判明 |
| 🛡️ 騎士 | 村人 | 毎夜1人を護衛 |
| 🥖 パン屋 | 村人 | 特殊能力なし |
| 🦊 妖狐 | 村人 | 特殊能力なし |
| 🤝 共有者 | 村人 | 特殊能力なし |
| 🐈 猫又 | 村人 | 特殊能力なし |
| 🐺 人狼 | 人狼 | 毎夜1人を襲撃 |
| 🐺 白狼 | 人狼 | 毎夜1人を襲撃 |
| 🃏 狂人 | 人狼 | 人狼の仲間（村人に偽装） |

## AI設定

- OpenAI / Gemini API Key（任意）を設定するとAIが本格的に発言・推理します
- 未設定でもサンプル発言でゲームを進行できます

## ファイル構成

```
index.html          # ロビーページ
game.html           # ゲームページ（BBS UI）
css/style.css       # スタイル
js/config.js        # 役職・定数定義
js/game.js          # ゲーム状態管理
js/ai.js            # AIプレイヤーロジック
js/bbs.js           # BBS レンダリング
js/main.js          # ロビーエントリーポイント
js/controller.js    # ゲームコントローラー
.github/workflows/deploy.yml  # GitHub Pages デプロイ
```

## デプロイ

`main` ブランチへのプッシュで GitHub Pages へ自動デプロイされます。
